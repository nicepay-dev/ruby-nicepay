# Nicepay API V1 Ruby Implementation

Folder ini berisi implementasi endpoint utama Nicepay API V1:
- Registration (Transaction Registration)
- Payment (Redirect ke Payment Page)
- Inquiry (Cek status transaksi)

Lihat dokumentasi resmi: https://docs.nicepay.co.id/nicepay-api-v1-professional

Setiap file Ruby di sini mengacu pada spesifikasi parameter dan response dari dokumen tersebut.
